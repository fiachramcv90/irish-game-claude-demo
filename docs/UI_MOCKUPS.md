# UI Mockups & Design Concepts

## 🎨 Design System Overview

### Color Palette

```
Primary Colors:
├─ Irish Green: #169B62 (main brand color)
├─ Warm Orange: #FF8C42 (action buttons, highlights)
├─ Sky Blue: #4ECDC4 (secondary actions, info)
└─ Sunset Purple: #9B59B6 (achievements, special content)

Neutral Colors:
├─ Cream White: #FFF9E6 (background, cards)
├─ Soft Gray: #F5F5F5 (borders, subtle elements)
├─ Dark Gray: #2C3E50 (text, icons)
└─ Light Green: #E8F5E8 (success states)

Child-Friendly Accents:
├─ Bright Yellow: #F1C40F (stars, celebrations)
├─ Coral Pink: #FF6B9D (hearts, positive feedback)
└─ Mint Green: #A8E6CF (calm, soothing elements)
```

### Typography

```
Primary Font: 'Nunito' (Rounded, friendly, highly legible)
├─ Headings: Bold, 24-32px
├─ Body Text: Regular, 16-18px
├─ Irish Text: SemiBold, 20-24px (emphasis on learning content)
└─ Buttons: Bold, 18-20px

Accessibility:
├─ Minimum contrast ratio: 4.5:1
├─ Large touch targets: 44px minimum
└─ Clear visual hierarchy
```
